## Background

This documentation was created with the Starlight framework. The actual content of these pages is located at `/docs/src/content/docs/`. If you are adding a new page, you will also have to update the applicable fields in `docs/astro.config.mjs`.

## Getting Started

### Setup 
```bash
cd docs && npm install
```

### Development
```bash
npm run dev
```

*The development server automtically reloads upon file changes
